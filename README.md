# capstone-project

A full-stack web application built with React, Node.js, Express, and MongoDB.

## Tech Stack

- **Frontend:** React (functional components, hooks)
- **Backend:** Node.js + Express
- **Database:** MongoDB (Mongoose)
- **Styling:** Tailwind CSS
- **Package manager:** npm

## Project Structure

```
/client          — React frontend (components, pages, hooks, assets)
/server          — Express backend (routes, controllers, models, middleware)
  /models        — Mongoose schemas
  /routes        — API route definitions
  /controllers   — Business logic for each route
.env             — Environment variables (never commit this file)
```

## Getting Started

> Setup instructions will be added once the client and server are scaffolded.

1. Clone the repository
2. Copy `.env.example` to `.env` and fill in your values
3. Install dependencies and run the dev servers (details coming soon)
